# Chatbot-de-Nutricao-com-IA-Generativa
Bot de Telegram desenvolvido para a disciplina de Inteligência Artificial que utiliza a API do Google Gemini para gerar dietas personalizadas. O sistema interpreta manuais clínicos e cria planos alimentares baseados nos dados do usuário (peso, altura, objetivos), funcionando como um assistente nutricional inteligente.
